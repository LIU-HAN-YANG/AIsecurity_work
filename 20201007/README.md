# Building matrices(建立矩陣)
```
diag(v[, k])
提取對角線或構造對角線數組。

diagflat(v[, k])
使用展平的輸入作為對角線創建二維數組。

tri(N[, M, k, dtype])
在給定對角線處及以下且在其他位置為零的數組。

tril(m[, k])
數組的下三角。

triu(m[, k])
數組的上三角。

vander(x[, N, increasing])
生成Vandermonde矩陣。
```
